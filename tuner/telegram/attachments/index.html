§Attachment Downloader and Storage Manager

§Overview

Checks incoming Telegram messages for files, images, or audio clips. Downloads them via HTTP and stores them inside the workspace’s telegram_files/ directory.

§Collaboration Graph

  • Integrates with process_text_with_files to inject paths into agent inputs.
